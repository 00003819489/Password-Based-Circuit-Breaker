# 🔐 Password Based Circuit Breaker (8051)

This project is a security-based system using 8051 microcontroller.

## 🚀 Features
- Password protected system
- Locks after 3 wrong attempts
- Correct password turns OFF LED

## 🛠️ Components
- AT89S52
- LCD Display
- Keypad
- LED / Relay
